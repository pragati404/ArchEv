# ArchEv
🔌 ArchEV – Autonomous EV Charging Robotic Arm
The ArchEV is an intelligent and cost-effective autonomous robotic arm system aimed at providing Electric Vehicle charging solutions. The system incorporates computer vision, depth perception, and robotic technology for precise Electric Vehicle charging port detection and connection without human intervention.
The system utilizes an RGB-D Camera (Intel RealSense) for real-time object detection and depth estimation for precise 3D positioning of the Electric Vehicle charging socket.

🚀 Key Features
🔍 Smart Detection – Electric Vehicle charging port detection using computer vision models
📏 Depth-Based Localization – Electric Vehicle charging socket positioning using Intel RealSense technology
🤖 Autonomous Robotic Arm – Precise Electric Vehicle charging socket positioning using computer vision
🔄 Universal Compatibility – Designed for compatibility with Electric Vehicle models
💰 Low-Cost Design – Emphasis on cost-effectiveness

🛠️ Technology Stack
Computer Vision – OpenCV/Deep Learning models
Intel RealSense – RGB-D Camera
Python
Robotics & Embedded Systems
Calibration & 3D Coordinate Mapping

🎯 Objective
To design an efficient and automated Electric Vehicle charging system for increased user convenience and future Electric Vehicle technology.
